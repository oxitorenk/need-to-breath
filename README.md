# Need To Breath

A minimalist web app to help pace smoking habits. It calculates when you are "allowed" to smoke next based on a configurable interval.

## Features

- **Pace Keeper**: Tracks time accumulated for your next cigarette.
- **Banking System**: Unused time stacks up as credits.
- **Simple Tracking**: One button to log a cigarette.
- **Themes**: Light and Dark mode support.
- **Offline Ready**: Works offline as a PWA.
